# 🌸 数学花园 - 快乐学算术

一个专为女孩设计的 100 以内加减法学习游戏，采用武侠风格的等级系统，完美适配 iPad 等平板设备。

## 📱 iPad/平板使用指南

### 方法一：添加到主屏幕（推荐）
像使用原生 App 一样方便：

**iPad 操作步骤：**
1. 用 Safari 打开游戏网页
2. 点击底部 **分享按钮**（方框带向上箭头）
3. 选择 **"添加到主屏幕"**
4. 点击 **"添加"**
5. 主屏幕上会出现 🌸 数学花园图标，点击即可全屏运行

**Android 平板操作步骤：**
1. 用 Chrome 打开游戏网页
2. 点击右上角 **菜单**（三个点）
3. 选择 **"添加到主屏幕"** 或 **"安装应用"**
4. 主屏幕会出现图标

### 方法二：全屏模式
- 点击开始界面的 **"📱 全屏模式"** 按钮
- 或点击 **"开始游戏"** 自动进入全屏

### 方法三：离线使用
首次加载后，游戏会自动缓存，**无需网络也能玩**！

## ✨ 特色功能

### 🎮 游戏特点
- **20道题/次**：每次练习20道题目，充分巩固知识
- **武侠等级系统**：从初出茅庐到数学至尊，共8个等级
- **混合运算**：高级等级支持加减混合运算（如：a + b - c）
- **计时功能**：记录每次游戏的总用时
- **可爱粉色主题**：符合女孩审美，花朵、星星、蝴蝶装饰
- **触摸友好**：专为手机和平板优化的数字键盘

### 🧠 智能学习系统
- **自动难度调整**：每3天分析一次学习表现
- **个性化推荐**：根据正确率自动建议升级/保持/降级
- **薄弱点识别**：自动分析加法、减法、混合运算的掌握情况
- **连续学习追踪**：记录学习天数，鼓励坚持

### 📊 进度追踪
- **实时统计**：答对数、连对数、正确率、总用时
- **历史记录**：保存最近50次游戏记录
- **学习报告**：详细分析强项和需要加强的地方

## 🎯 武侠等级系统

| 等级 | 称号 | 数字范围 | 运算类型 | 难度 |
|------|------|----------|----------|------|
| 1 | 初出茅庐 | 1-10 | 加法 | ⭐ |
| 2 | 江湖新秀 | 1-20 | 加减法 | ⭐ |
| 3 | 小有名气 | 1-30 | 加减法 | ⭐⭐ |
| 4 | 武林高手 | 10-50 | 加减法、加减混合 | ⭐⭐ |
| 5 | 一代宗师 | 1-70 | 加减法、加减混合 | ⭐⭐⭐ |
| 6 | 绝世奇才 | 10-80 | 加减法、加减混合 | ⭐⭐⭐ |
| 7 | 独孤求败 | 20-90 | 加减法、混合运算、连加连减 | ⭐⭐⭐⭐ |
| 8 | 数学至尊 | 1-100 | 所有运算类型 | ⭐⭐⭐⭐⭐ |

### 运算类型说明
- **加法**：a + b（如：8 + 9 = 17）
- **减法**：a - b（如：15 - 7 = 8）
- **加减混合**：a + b - c（如：5 + 8 - 3 = 10）
- **连加**：a + b + c（如：4 + 5 + 6 = 15）
- **连减**：a - b - c（如：20 - 5 - 3 = 12）

## 🚀 部署方式

### 方式一：直接打开
用浏览器打开 `index.html` 文件即可

### 方式二：本地服务器
```bash
cd math-game
python3 -m http.server 8000
# 访问 http://localhost:8000
```

### 方式三：部署到网络（推荐）
上传到静态网站托管服务：
- **GitHub Pages**（免费）
- **Netlify**（免费）
- **Vercel**（免费）
- 腾讯云/阿里云 OSS

**GitHub Pages 部署步骤：**
```bash
cd math-game
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/math-game.git
git push -u origin master
```
然后在 GitHub 仓库 Settings → Pages 中开启。

## 🎨 平板优化特性

### 触控优化
- **大按键设计**：数字键盘按钮最小 60px，适合手指点击
- **触控反馈**：按钮按下时有明显的视觉反馈
- **防误触**：禁用双击缩放，避免游戏时意外放大

### 响应式布局
- **iPad 竖屏**：优化为 600px 宽度，字体更大
- **iPad 横屏**：自适应布局，题目显示更清晰
- **iPad Pro**：支持大屏显示，按钮更大更易点

### PWA 功能
- **离线运行**：首次加载后无需网络
- **添加到主屏幕**：像原生 App 一样启动
- **全屏模式**：沉浸式游戏体验，无浏览器干扰
- **独立应用**：从主屏幕启动时，不显示浏览器地址栏

## 📝 数据存储

所有数据保存在设备本地：
- 游戏进度：`localStorage.mathGameProgress`
- 分析数据：`localStorage.mathGameAnalytics`

**注意**：
- 数据不会自动同步到其他设备
- 清除浏览器数据会丢失进度
- 卸载 App（从主屏幕删除）不会丢失数据，重新添加即可恢复

## 🎁 升级规则

系统每3天自动分析学习情况：
- **正确率 ≥ 85%**：建议升级到下一级
- **正确率 50%-85%**：保持当前等级继续练习
- **正确率 < 50%**：建议降低等级巩固基础
